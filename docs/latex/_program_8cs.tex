\doxysubsection{Detailed Description}
Application startup and host configuration for PC Part Picker. 

High-\/level responsibilities:\+
\begin{DoxyItemize}
\item Configure services (Db\+Context, \doxylink{interface_project__6_______group__4_______c_s_c_n73060___s_e_c__1_1_1_services_1_1_i_part_service}{IPart\+Service}, CORS, Swagger)
\item Configure JSON serialization options for API controllers
\item Apply EF Core migrations and run $<$\+Db\+Seeder$>$ at startup
\item Serve SPA Client\+App static files and provide API endpoints
\end{DoxyItemize}

This file is the entry point for the web application and is important to document so developers know where migrations, seeding, CORS and static file hosting are configured.
